AI Fitness Intelligence System 🤖💪

A fully deployed AI-powered fitness web application that predicts calories burned, analyzes user data, and provides intelligent fitness and lifestyle recommendations.

Overview

This project implements a complete end-to-end data science pipeline, transforming raw fitness data into actionable insights through machine learning and interactive visualization.

It integrates:

- Machine Learning for prediction
- Data Analysis for insights
- Rule-based logic for recommendations
- Web deployment for real-time interaction

---

Key Features

- Calorie prediction using machine learning models
- BMI calculation and health classification
- Exercise-based calorie estimation
- Personalized diet recommendations
- Fitness scoring based on lifestyle inputs
- Future weight prediction system
- Interactive dashboard with visual analytics
- User authentication system

---

Machine Learning Pipeline

- Data preprocessing and cleaning
- Feature engineering (Intensity calculation)
- Model training:
  - Linear Regression
  - Decision Tree Regressor
- Model evaluation using R² Score and Mean Absolute Error
